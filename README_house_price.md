# House Price Predictor

A beginner Machine Learning project that predicts house prices based on area, bedrooms, and age using Linear Regression.

##  What it does??
- Takes house features as input (area, bedrooms, age)
- Predicts the price in lakhs
- Plots Actual vs Predicted prices

##  Tech Stack:
- Python
- Scikit-learn
- Pandas, NumPy
- Matplotlib

##  How to Run??

```bash
pip install scikit-learn pandas numpy matplotlib
python house_price_predictor.py
```

##  Output:
- R² Score and RMSE printed in terminal
- Scatter plot saved as `house_price_plot.png`

##  Real World Use:
Real estate platforms like MagicBricks and 99acres use similar models to estimate property prices.

## Concepts Used:
- Linear Regression
- Train-Test Split
- Model Evaluation (R², RMSE)
